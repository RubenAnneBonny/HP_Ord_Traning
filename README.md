# HP Ord Träning

A web app for practicing the ORD (vocabulary) section of the Swedish university aptitude test (Högskoleprovet).

## Prerequisites

1. **Node.js** (version 18 or later) — download from [nodejs.org](https://nodejs.org/)
   - This also installs `npm`, which you need to install dependencies

## Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/RubenAnneBonny/HP_Ord_Traning.git
cd HP_Ord_Traning
npm install
```

## Running the app

Start the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Other commands

| Command | Description |
|---|---|
| `npm run dev` | Start development server with hot reload |
| `npm run build` | Build for production |
| `npm run start` | Run the production build |
| `npm run lint` | Run the linter |
